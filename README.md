# Cryptnox Homebrew Tap

Welcome to the official Homebrew Tap for Cryptnox CLI tools and software. This repository provides an easy way to install and manage Cryptnox tools using Homebrew.

## Available Formulae

- `cryptnox/tools/cryptnox-cli`: The Cryptnox Command Line Interface (CLI) tool.

## Installation

To use this tap, first add it to your Homebrew setup:

```bash
brew tap cryptnox/tools
```

Then, install the `cryptnox-cli` formula:

```bash
brew install cryptnox-cli
```

Or install it directly:
```bash
brew install cryptnox/tools/cryptnox-cli
```
